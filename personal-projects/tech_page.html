JavaScript Syntax

Data Types

There are seven data types if JavaScript, which are classiffcations for different kinds of data uses in programming.

  1. Number: Any number, including numbers with decimals
  2. String: Any grouping of characters on your keyboard (letters, numbers, spaces, symbols, etc.) surrounded by single quotes: '...'
  3. Boolean: Only 2 possible values, true or false (without quotes). Like off and on switches or answers to 'yes' or 'no' questions
  4. Null: Represents the intentional absence of value, and represented by the keyword null (without quotes)
  5. Undefined: Keyword undefined represents the absence of a value. Has a different use than null
  6. Symbol: Unique indentifiers, useful in more complex coding
  7. Object: Collections of related data
  8. Arithmatic Operators:

    9. operator: a character that performs a task in our code

    • Add +
    • Subtract -
    • Multiply *
    • Divide /
    • Remainder %

String Concatenation

When + is used on two strings, it appends the right string to the left string:

console.log('one' + ',' + 'two' + ',' + 'three!') //Prints one, two, three!

Properties

Properties are the values associated with a JavaScript object. A characteristic of an object, often describung attributes associated with data structure. Every string instance has a property.

There are two kinds of properties:

  • Instance properties hold data that are specific to a given object instance.
  • Static properties hold data that are shared among all object instances.

A property has a name (a string) and a value (primitive, method, or object refrence).

Every string instance has a property called length that states that number of character int that string. . is another operator called dot operator

console.log('Hello'.length); //Prints 5

Method

Methods are actions we can preform on objects. Methods are properties containing a function definition. They are functions stored as object properties.

  • Call, or use, these methods by appending an instance with:
    • a period (dot opertator)
    • the name of the method
    • opening and closing parentheses
'example string'.methodName()

Create a Variable: var

Variables can hold data values and can be changed anytime. You can assign a value to a variable using equal to (=) operator when you delacre it of before using it. Can use var or let and const to create, or declare, variables.

var myName = 'Arya';
console.log(myName); //Prints Arya
  1. var = variable, declare
  2. myName = variable name (camel cap)
  3. = assignment operator, assigns value ('Arya') to variable (myName)
  4. 'Arya' = value assigned
  5. After declared, string value 'Ayra' is printed to console

There are general rules for naming variables, best practices to keep your code organized.

  • Can't start with numbers
  • Case senseitive, so myname and myName would be two different variables
  • Cannot be the same as keywords

Though you can use var to declare a variable, it is best practice to use either lets or const instead.

  • lets: signals that the variable can be reassigned a different value
    • let meal = 'Enchiladas';
    console.log(meal); // Prints Enchiladas
    meal = 'Burrito';
    console.log(meal); // Prints Burrito
    • Can declare a variable without assigning a value to the variable. Will automatically initialized with a value of undefined
    let price;
    console.log(price); // Print undefined
    price = 350;
    console.log(price); // Print 350
  • const: constant, cannot be reassigned and must be assigned a value
    • const myName = 'Gilberto';
    console.log(myName); // Print Gilberto

Together with variables and Mathimatical Assignment Operators, you can run a few simple built-in arguments.

  • Mathamatical Assignment Operators
    • let w = 4;
    w = w + 1;
    console.log(w); //Output: 5
  • Built-in Mathamatical Assignment Operators
    • -=
    • *=
    • /=
    let w = 4;
    w += 1;
    console.log(w); //Output: 5
  • Increment and Decrement Operators
    • (++) Increment = increase var by 1
    • (--) Decrement = decrease var by 1
    let a = 10;
    a++; //Output: 11
    let b = 10;
    b--; //Output: 9

String Concatenation with Variables

Concatenation means to link (things) together in a chain or series. With JavaScript, we are able to do this with strings and variables, called String Concatenation.

To concatenate a string, you add a plus sign (+) between the strings or variables you want to connect. Anything like a space or punctuation (period, comma, ect) must be placed within the quotes of the string to appear.

let myPet = 'armadillo';
console.log('I own a pet ' + myPet + '.'); //Output: I own a pet armadillo.

String Interpolation

Similar to string concatenation, String Interpolation inserts(interpolate) variables into strings using template literals. A template literal is a way to concatenate strings while allowing embedded expressions and improving readablilty.

  • template literals are wrapped up in brackets, the button to the left of 1 on your keyboard (` `)
  • Within the brackets is the string and the placeholder for the variable you're inserting `${VarName}`
const myPet = 'armadillo';
console.log(`I own a pet ${myPet}.`); //Output: I own a pet armadillo.

typeof Operator

One last thing we can do with JavaScript syntax is to figure out the data type of a value. With the typeof operator, we can check the value to its right and return (or pass back) a string of the data type. Some examples:

const unknown1 = 'foo';
console.log(typeof unknown1); // string
const unknown2 = 10;
console.log(typeof unknown2); // number
const unknown3 = true;
console.log(typeof unknown3); // boolean

Conditional Statments

A Conditional Statment checks a specific condition(s) a preforms a tacks based in that condition(s).

If Statement

if(true) {
console.log('This message will print!');
}
  • if keyword followed by () followed by code block
  • Inside parentheses (), a condition is provided that evaluates to true or flase
  • If condotion evaluates true then code in curly braces {} executes
  • If false code won't run

If...Else Statment

if (false) {
console.log('The code in this block will not run.');
} else {
console.log('But the code in this block will!');
}
  • else keyword following the code block of an if statment
  • Has code block that is wrapped by a set of curly braces {}
  • Code in else statment will execute when that of statement's condition evaluates to false

Logical Operators

Operators that work with boolean values

  • the and operator - (&&)
  • the or operator - (||)
  • the not operator, otherwise known as the bang operator - (!)

&&: checks if two things are true; Both need to be true for the entire condition to be true and execute.

if (spotlight === 'green' && pedestrians === 0) {
console.log('Go!');
} else {
console.log('Stop!');
}

||: Checks if either condition is true; one must be true, else BOTH will execute as false.

if(day === 'Saterday' || day === 'Sunday'){
console.log('Enjoy the weekend!');
} else {
console.log('Do some work!');
}

!: Not op reverses, or negates, the value of a boolean. Either passes true value back to false or a false value back to true.

let excited = true;
console.log(!excited); // false
let sleepy = false;
console.log(!sleepy); // true
